## 有效的括号

有效字符串需满足：

- 左括号必须用相同类型的右括号闭合。
- 左括号必须以正确的顺序闭合。
- 注意空字符串可被认为是有效字符串。

`思路`: 使用 `栈`的思想，遇到一个左括号就入栈，遇到右括号就出栈，然后判断是否是一组括号

```javascript
  var VaildKuoHao = function (str) {
    var map = {
      '(' : -1,
      ')' : 1,
      '{' : -2,
      '}' : 2,
      '[' : -3,
      ']' : 3
    }
    
    var stack = []
    
    for (let i = 0; i < str.length; i++) {
      if (map[str[i]] < 0) {
        // 左括号，入栈
        stack.push(map[str[i]])
      } else {
        // 右括号，出栈
        let last = stack.pop()

      }
    }
    
    if (stack.length > 0) {
```
