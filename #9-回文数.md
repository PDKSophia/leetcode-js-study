## 回文数

判断一个整数是否是回文数。回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。

示例 1:

输入: 121
输出: true
示例 2:

输入: -121
输出: false
解释: 从左向右读, 为 -121 。 从右向左读, 为 121- 。因此它不是一个回文数。
示例 3:

输入: 10
输出: false
解释: 从右向左读, 为 01 。因此它不是一个回文数。


```javascript

/**
 * desc: 第一种方法，将数字转为字符串
 * @param {number} x
 * @return {boolean}
 */
var isPalindrome = function(x) {
    var arr = []
    var str = x.toString()
    for(var i = str.length-1; i>=0; i--) {
        arr.push(str[i])
    }

    if(x.toString() === arr.join('')) {
        return true
    } else {
        return false
    }
};

```

```javascript

/**
 * desc: 第二种方法，使用栈
 * @param {number} x
 * @return {boolean}
 */
var isPalindrome = function(x) {
   var arr = []
   for (let i = 0; i < x.length; i++) {
     arr.push(x[i])
   }
   
   var result = ''
   while (arr.length > 0) {
     result += arr.pop()
   }
   
   return result
};

```

```javascript

/**
 * desc: 第三种方法
 * @param {number} x
 * @return {boolean}
 */
var isPalindrome = function (x) {
  let strx = x.toString()
  return x == strx.split('').reverse().join('')
}

```
